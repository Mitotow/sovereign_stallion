# Sovereign Stallion
## Récupération des sources pour développement
Récupération du dépôt
```bash
```bash
git clone ...
```
Se rendre dans le projet
```bash
cd sovereign_stallion
```
Créer un environnement virtuel python
```bash
python -m venv sovereign_stallion
```
Activer l'environnement
- Windows : `.\sovereign_stallion\Scripts\activate`
- Mac/Linux : `source sovereign_stallion/bin/activate`
Installer les dépendances
```bash
pip install -r requirements.txt
```
